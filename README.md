## Getting Started

### 1. Installation

#### Clone AI-ADC Repository

```bash
cd ~
git clone https://github.com/Kutsev-Ozyoruk/AI-ADC
cd AI-ADC
```

#### Prerequisities

You will need to build the AI-ADC environment by following commend:

```bash
pip3 install -r requirements.txt
```
